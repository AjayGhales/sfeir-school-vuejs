# Bootstrapping de l'application
<br><br>

```bash
npm install -g @vue/cli
```
<br>
```bash
npm run client
```
<br>
```bash
npm run server
```
<br>
<p class="center">
<span><strong>npm run client:</strong></span>
<span> http://localhost:8080/<span>
<br>
